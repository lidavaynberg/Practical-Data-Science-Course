## â“ Common Questions
**Q: Do I need to know git commands?**
A: No! You can use GitHub's web interface for all basic tasks.

**Q: What should go in my README?**
A: Project purpose, data sources, instructions, and a brief impact statement.

**Q: How do I use Copilot or ChatGPT?**
A: Paste your code and ask for an explanation or suggestions. Include the AI's response in your repo for documentation.

**Q: Can I collaborate with classmates?**
A: Yes! You can add collaborators to your repo or fork each other's projects.
